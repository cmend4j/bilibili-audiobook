B站有声书工具 v2.0.0
========================

功能：下载B站合集音频 → 按章节切割 → 变速处理 → 批量下载


文件说明
--------
  B站有声书工具.exe   打包好的可执行文件，双击运行
  bilibili_audiobook.py   Python源码（需要Python环境才可用）
  ffmpeg.exe          音频处理引擎（必需，与exe放同一目录即可）
  config/             配置文件目录（自动创建）


使用前准备
----------
  1. 确保 ffmpeg.exe 与 B站有声书工具.exe 在同一目录下
  2. 双击 B站有声书工具.exe 即可启动，无需安装任何Python环境


五个标签页
----------
  标签页1「下载与切割」
    输入B站有分P合集的视频链接 → 下载音频 → 按固定时长切章
    支持两种切割模式：
      · 固定时长：每N分钟切一章（默认20分钟）
      · 章节对齐：配合小说TXT文件，用Whisper识别定位章节边界（需安装openai-whisper）

  标签页2「变速处理」
    选择已有章节文件夹 → 设置倍速(0.5x~3.0x) → 输出加速后的MP3
    · 倍速可用滑块拖动，也可精确输入数值
    · 输出可选每章独立 或 每N章合并
    · 使用atempo滤镜，变速不变调

  标签页3「日志」
    实时显示所有操作的运行日志
    · 支持清空和保存日志

  标签页4「批量处理」
    批量下载多个B站合集
    · 支持手动添加URL或从文件导入
    · 任务队列管理（添加、移除、清空）
    · 实时显示下载进度和状态
    · 自动保存队列，下次启动可继续

  标签页5「历史记录」
    查看和管理历史下载记录

注意事项
--------
  · ffmpeg.exe 约84MB，是从系统提取的，与exe放同目录即可便携使用
  · 如果移动文件夹，请确保 ffmpeg.exe 和 exe 一起移动
  · 下载功能需要网络连接，B站链接必须是有分P的合集
  · 章节对齐模式需要额外安装：pip install openai-whisper
  · 配置文件保存在 config/ 目录下
  · 批量队列文件保存在 config/batch_queue.json


源码运行
--------
  如果想用源码运行（需要Python 3.8+）：
    pip install yt-dlp
    python bilibili_audiobook.py
