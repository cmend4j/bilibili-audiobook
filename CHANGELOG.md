# 更新日志

所有 notable changes 都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [2.0.0] - 2026-06-15

### 新增
- 批量下载功能：支持批量URL导入和队列管理
- 真实进度显示：下载进度实时更新
- 配置持久化：用户设置自动保存到JSON文件
- 历史记录：记录下载历史
- 取消功能：支持取消正在进行的任务
- 日志系统：支持文件日志记录
- 输入验证：更严格的URL和路径验证

### 改进
- 错误处理：细化异常类型，区分FileNotFoundError、PermissionError等
- 类型注解：为所有函数添加参数和返回值类型提示
- 速度范围：扩展到0.5x-3.0x
- 进度条：从indeterminate改为determinate模式
- 代码质量：提取魔法数字为常量，改善代码风格

### 修复
- 修复裸except捕获所有异常的问题
- 修复路径处理兼容性问题
- 修复多个潜在的资源泄漏问题

## [1.0.0] - 2026-06-07

### 新增
- 基础下载功能：下载B站分P合集音频
- 章节切割：按固定时长切割音频文件
- 变速处理：支持变速不变调
- GUI界面：tkinter图形界面
- PyInstaller打包：支持打包为单个exe文件
